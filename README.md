# AI_Course_Live_Project_1 
A simple AI chatbot built using Python, Streamlit, and the Google Gemini API. Users can enter a question and receive an AI-generated response in a clean, interactive chat interface.

Features

- Simple and user-friendly interface
- Powered by the Google Gemini API
- Secure API key management using a ".env" file
- Chat-style conversation interface
- Beginner-friendly project structure

Project Structure

Gemini-Chatbot/
│── app.py
│── requirements.txt
│── .env.example
│── .gitignore
│── README.md

Installation

1. Clone the repository:

git clone https://github.com/Tanvi566/AI_Course_Live_Project_1.git

2. Move into the project folder:

cd Gemini-Chatbot

3. Install the required packages:

pip install -r requirements.txt

4. Create a ".env" file and add your Gemini API key:

GEMINI_API_KEY=your_gemini_api_key_here

Running the Application

Start the Streamlit app by running:

streamlit run app.py

The chatbot will open in your default web browser.
